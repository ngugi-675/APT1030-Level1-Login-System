# Level 2: SACCO Member Financial Record

## Overview
A program that manages SACCO member financial records, demonstrating static vs dynamic typing concepts.

## Features
- Store member information (Name, ID)
- Record 6 months of contributions
- Calculate total savings
- Type coercion demonstration

## Files in this Level
- `sacco.py` - Python implementation (dynamic typing)
- `SaccoMember.java` - Java implementation (static typing)
- `TYPE_TESTING.md` - Analysis of type system behaviors

## Quick Start

### Python
```bash
python sacco.py
