# Level 1: Login Validation System

## Overview
Multi-language implementation of a login validation system for eCitizen Kenya.

### Languages Used
- Python 3.x
- Java 17+
- C++11/14

## Features
- Username/password validation
- Predefined credentials (adminKE / 254Secure)
- Access Granted/Denied messaging

## Quick Start

### Python
```bash
python login.py
